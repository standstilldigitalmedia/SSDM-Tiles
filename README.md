## Part of the SSDM Ecosystem

SSDM-UI is one of several addons in the Standstill Digital Media ecosystem:

- **[SSDM-Core](https://github.com/standstilldigitalmedia/SSDM-Core)** - Shared types and utilities all other addons depend on
- **[SSDM-UI](https://github.com/standstilldigitalmedia/SSDM-UI)** - Base animated UI elements
- **[SSDM-Manager](https://github.com/standstilldigitalmedia/SSDM-Manager)** - Manager pattern for resource and scene management
- **SSDM-Mesh** - Mesh combining and texture atlas building logic
- **SSDM-Tiles** *(this addon)* - 3D tile mapping system
- **SSDM-AWOC** - Avatar Wardrobe Organizer and Colorer

## Installation

### Prerequisites

SSDM-Tiles requires [SSDM-Core](https://github.com/standstilldigitalmedia/SSDM-Core), [SSDM-Manager](https://github.com/standstilldigitalmedia/SSDM-Manager),
and [SSDM-UI](https://github.com/standstilldigitalmedia/SSDM-UI). Install these first.

### Via Asset Library (Recommended)
1. Open Godot Editor
2. Go to **AssetLib** tab
3. Search for "SSDM-Core" and install it
4. Search for "SSDM-Tiles" and install it

### Manual Installation
1. Download [SSDM-Core](https://github.com/standstilldigitalmedia/SSDM-Core) and copy `addons/SSDM-Core/` to your project's `addons/` directory
2. Download [SSDM-Manager](https://github.com/standstilldigitalmedia/SSDM-Manager) and copy `addons/SSDM-Manager/` to your project's `addons/` directory
3. Download [SSDM-UI](https://github.com/standstilldigitalmedia/SSDM-UI) and copy `addons/SSDM-UI/` to your project's `addons/` directory
4. Download [SSDM-Tiles](https://github.com/standstilldigitalmedia/SSDM-Tiles) and copy the `addons/SSDM-Tiles/` folder to your project's `addons/` directory
